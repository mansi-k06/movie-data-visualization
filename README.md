# 🎬 Movie Data Analysis — Matplotlib & Seaborn

A data visualization mini project built using Python, Matplotlib, and Seaborn.

## 📌 About
This project explores a synthetically generated IMDB-style movie dataset 
(300 movies, 1990–2024) through 5 key visualizations.

## 📊 Visualizations Included
1. 🎬 Genre Popularity — Horizontal Bar Chart
2. ⭐ Ratings Distribution — Histogram with KDE
3. 🎥 Runtime vs Rating — Scatter Plot by Genre
4. 📅 Movies Released Per Year — Line Chart
5. 🔥 Director vs Genre — Heatmap of Avg Ratings

## 🛠️ Libraries Used
- Python 3
- Pandas
- NumPy
- Matplotlib
- Seaborn

## ▶️ How to Run
Open the `.ipynb` file in Google Colab (Recommended) or Jupyter Notebook
